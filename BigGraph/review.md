# Review

## tools

一个用于生成图的命令行工具，相关命令如下：

```c++
usage:
text2bin textfilename binfilename: text file to binary file conversion
info filename: print graph metadata
print filename: print graph topology (careful with big graphs)
noout filename: detect vertices with no outgoing edges
noin filename: detect vertices with no incoming edges
edgestats filename: print stats on graph edges: e.g., min/max edges per node, etc.
```
